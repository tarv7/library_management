üìã Users Controller Documentation

Controller: Api::V1::UsersController
Description: Manages user creation in the library management system
POST /api/v1/users

Description: Creates a new user (member or librarian) in the system

Request Body:

{
  "user": {
    "email_address": "user@example.com",
    "password": "password123",
    "password_confirmation": "password123",
    "name": "John Doe",
    "role": "member"  // or "librarian"
  }
}

Success Response (201):

{
  "id": 1,
  "email_address": "user@example.com",
  "name": "John Doe",
  "role": "member"
}

Error Response (422):

{
  "email_address": ["can't be blank", "has already been taken"],
  "password": ["is too short (minimum is 6 characters)"],
  "name": ["can't be blank"],
  "role": ["is not included in the list"]
}
